# JOB-131 — Stack Growth Solutions AI Integration Engineer

Reference scaffold for the Upwork application. Production platform pattern for senior
automation + AI integration across PE-backed finance back offices.

This repo is a **reference scaffold** (not a deployed application). The actual integration
work for Stack Growth Solutions would be done in their private client repos.

- `SPEC.md` — full technical spec (7 sections)
- `PROPOSAL.md` — answers to all 5 mandatory Upwork questions with prior-engagement examples
- `COVER_LETTER.txt` — cover letter (under 5,000 char Upwork limit, starts with `PIPELINE`)
- `README.md` — Business Problem Solved + 6 capability pillars + acceptance criteria
- `app/` — Python reference scaffold (LangGraph + FastAPI + Pydantic + n8n adapter)
- `tests/` — pytest suite
- `diagrams/` — 4 white-bg SVG diagrams (architecture, multi-agent, workflow, project-structure)
- `.planning/` — CONTEXT.md + RESEARCH.md + 7 GSD-style phase plans
- `docs/PROJECT_OVERVIEW.md` — scope, success criteria, stakeholder model
- `BUILD_OUTCOME.md` — build outcome + verification log
- `ROADMAP.md` — phased rollout plan
- `CLAUDE.md` — AI-assistant context
- `job.json` — machine-readable job metadata
- `OUT_OF_SCOPE.md` — explicit non-goals
